Значки

Значки могут быть трёх основных типов:
  • Ярлык/Команда
  • Разделитель
  • Плагин (он же Доклет или Docklet, в дальнейшем для удобства я буду просто называть их плагинами).
Примечание. На данный момент реализована поддержка лишь немногих доклетов от ObjectDock (по причинам пока неясным автору программы).

Добавить значок или вкладку можно через пункт всплывающего меню 'Add ...' либо перетаскиванием из проводника (или подобных ему программ таких как Total Commander).

Удалить значок можно перетащив его за пределы панели и нажав в появившемся меню пункт "Удалить значок".
Если этого не сделать, то значок останется на рабочем столе. В этом случае в свойствах значка можно настроить его размер.

Изменить порядок следования вкладок или их названия можно в 'Редакторе значков и вкладок' который доступен во всплывающем меню панели (пока это единственный способ).

Включить отображение вкладок можно на первой странице в Настройках панели (при первом запуске программы отображение вкладок отключено).

 Примечание. Кликом средней кнопки мыши (колесико) на любом значке можно вызвать стандартное меню панели.

Команды

Встроенные команды можно использовать в ярлыках либо в 'Окне команд'. Окно команд вызывается нажатием тильды (~) когда панель активна (например, после клика на ней или на любом значке).

Команды можно выполнять группами, для этого их следует разделить символом ';' (точка с запятой). В таких пакетных командах встроенные команды (/hide, /exwin и т.п.) можно комбинировать с командами операционной системы (запуск программ, открытие файлов).

Примечание. Далее все команды приведены в виде имя_команды(параметры). В таком виде они используются в 'Окне команд'. В свойствах ярлыка имя команды пишется отдельно в поле ввода команды, а параметры (если имеются), через запятую и без скобок, в поле ввода параметров (см. пример ниже).

Список команд

/quit - закрыть программу (команда доступна через меню).

/hide - скрыть панель.

/options - открыть настройки прогарммы (команда доступна через меню).

/menueditor - показать редактор значков и вкладок (команда доступна через меню).

/site(новое_расположение) - изменить расположение панели. Возможные значения параметра: left - слева, right - справа, top - сверху, bottom - снизу.

/autohidetabs - переключить автоматическое скрытие вкладок в противоположный режим (равносильно снятию/установке метки "Только при наведении" в настройках панели).

/exwin(действие) - завершить работу системы. Возможные действия: 0 - завершение сеанса пользователя (выход из системы); 1 - выключение компьютера; 2 - перезагрузка; 4, 5 и 6 - то же, но в форсированном режиме. В форсированном режиме система не ожидает закрытия всех программ, а при необходимости принудительно завершает их.

/taskmgr - запуск Диспетчера задач (команда доступна через меню).

/suspend(Hibernate, ForceCritical, DisableWakeEvent) - включить спящий режим. Для всех параметров возможные значения 0 или 1. Пример вызова: /suspend(1,0,0) - гибернация, /suspend(0,0,0) - сон.

/setdisplaymode(ширина,высота,глубина_цвета,частота_обновления_экрана) - установить параметры экрана. Все параметры ограничены возможностями конкретного монитора. Глубина цвета - 8,16 или 32 (стандартным значением является 32 бит/пиксел). Пример вызова: /setdisplaymode(1024, 768, 32, 85).

/bsm(uMsg,wParam,lParam) - послать широковещательное системное сообщение (Для опытных пользователей. Имя команды - сокращение от BroadcastSystemMessage).

/windowfrompoint - возвращает дескриптор окна под курсором мыши (помещает его во внутреннюю переменную и в буфер обмена).

/findwindow(имя_класса_окна,имя_окна) - ищет дескриптор окна и помещает его во внутреннюю переменную. Поиск производится по любому одному, либо по совокупности параметров.

/findwindowex(имя_класса_окна,имя_окна) - ищет дескриптор дочернего окна и помещает его во внутреннюю переменную . Поиск производится по любому одному, либо по совокупности параметров. Дескриптор родительского окна должен содержаться во внутренней переменной.

/setwindowtext(текст) - изменяет заголовок окна. Значение дескриптора окна берется из внутренней переменной.

/sendmessage(uMsg,wParam,lParam) - послать сообщение окну (только для опытных пользователей). uMsg - числовой код сообщения, wParam и lParam - параметры. Значение дескриптора окна берется из внутренней переменной.

/lwa(enable,alpha) - установить окну режим и величину прозрачности. Параметр 'enable' принимает значения 0 (выключить режим прозрачности) и 1 (включить режим прозрачности). Параметр 'value' принимает значения от 0 (полная прозрачность) до 255 (полная непрозрачность). Значение дескриптора окна берется из внутренней переменной.  

/cmd - показать 'Окно команд'.

/opentab(имя_вкладки) - открыть вкладку по имени.

/opentabindex(номер_вкладки) - открыть вкладку по номеру.

/killprocess(имя_процесса.exe) - принудительно завершить процесс по имени.

/wacmd(код_команды) - послать команду проигрывателю Winamp. См. коды команд ниже.

/opencd(буква_диска) - открыть определенный привод CD/DVD. Параметр - буква требуемого диска.

/closecd(буква_диска) - закрыть определенный привод CD/DVD. Параметр - буква требуемого диска.

/play(имя_файла) - воспроизвести звуковой файл.


Числовые коды команд проигрывателя Winamp
  • 40044 - предыдущий трэк
  • 40045 - кнопка 'Play'
  • 40046 - кнопка 'Pause'
  • 40047 - кнопка 'Stop'
  • 40048 - следующий трэк
  • 40001 - закрыть Winamp
  • 40012 - открыть настройки
  • 40029 - диалог открытия файла
  • 40154 - в начало плэй-листа
  • 40157 - остановка после окончания проигрываемого трэка
  • 40158 - в конец плэй-листа
  • 40192 - запуск визуализации
Пример

Для того чтобы убрать назойливую рекламу в Word 2010 Starter создайте значок со следующими параметрами

команда:            /findwindow;/findwindowex;/sendmessage;/findwindow;/findwindowex;/showwindow
параметры:    OpusApp;MsoCommandBarDock,MsoDockRight;16;OpusApp;_WwF;3

или выполните команду в 'Окне команд' (вызывается по нажатию тильды ~)

/findwindow(OpusApp);/findwindowex(MsoCommandBarDock,MsoDockRight);/sendmessage(16);/findwindow(OpusApp);/findwindowex(_WwF);/showwindow(3)

Подстановочные константы

Подстановочные костанты могут помочь сохранить работоспособность созданных вами ярлыков файлов и папок при:
  • переносе Терри вместе с вашим файлом настроек на другой компьютер
  • изменении некоторых параметров настройки вашего компьютера
  • переустановке операционной системы
  • использовании Терри на переносном носителе данных.
Список констант:
  • %pp% - папка Терри
  • %pf% - папка Program Files текущей операционной системы (обычно C:\Program Files)
  • %doc% - папка "Мои документы"
  • %desktop% - папка "Рабочий стол"
  • %shell% - текущий шелл (из Настроек программы)
  • %windir% - корневая папка системы (обычно C:\Windows)
  • %systemroot% = %windir%
  • %sysdir% - системная папка (обычно C:\Windows\System32)
Все константы заменяются на путь без завершающего знака косой черты.

Пример
  • %pp%\default.png (часть '%pp%\' можно опустить)
  • %pf%\total\totalcmd.exe
Примечание. При создании пунктов меню (перетаскиванием файлов и папок из проводника) или при выборе файлов и папок в диалоге выбора файла фрагменты пути будут автоматически заменены соответствующей константой при обнаружении совпадений.
