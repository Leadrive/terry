Значок "Корзина".
/quit
Выйти из программы.
/hide
Полностью скрыть программу. Показать ее можно используя горячие клавиши (настраивается на странице "Общие" в Настройках программы).
/say(Message text)
Вывести сообщение с текстом "Message text".
/alert(Alert message!)
Вывести тревожное сообщение с текстом "Alert message!".
/sets
Открыть окно настроек программы. В параметрах указывается номер открываемой вкладки (по-умолчанию первая вкладка). Нумерация идет с нуля.
/cmd
Открыть окно ввода команд.
/collection
Открыть коллекцию иконок.
/apps
Открыть список установленных приложений.
/taskmgr
Запустить диспетчер задач.
/program
Добавить программу на Терри. Будет показан диалог выбора файла для выбора исполняемого файла программы.
/command
Открыть это окно (окно выбора команд).
/backup
Сделать резервную копию файла настроек.
/restore
Восстановить файл настроек из резервной копии.
/tray
Показать окно области уведомлений.
/autotray
Включить/отключить отображение всех значков и уведомлений на панели задач (необходимо отключить для работы команды /tray).
/togglesystaskbar
Скрыть/показать системную панель задач.
/themeeditor
Открыть редактор текущего стиля
/lockdragging
Запретить/разрешить перетаскивание значков
/site(bottom)
Переместить программу к указанному (в качестве параметра) краю экрана. left - слева, top - сверху, right - справа, bottom - снизу
/logoff
Завершить сеанс пользователя.
/shutdown
Выключить компьютер.
/reboot
Перезагрузить компьютер.
/suspend
Перевести компьютер в ждущий режим.
/hibernate
Перевести компьютер в спящий режим.
/killprocess(explorer.exe)
Убить процесс по имени (имя в параметрах).
/setdisplaymode(1024,768,32,60)
Установить параметры дислея (ширина, высота, глубина цвета, частота).
/bsm(18,0,0)
Послать широковещательное сообщение (номер сообщения "WM_...", параметр WPARAM, параметр LPARAM).
/windowfrompoint
Поместить числовой дескриптор окна во внутреннюю переменную.
/findwindow(Button,Start)
Найти окно по (имя класса, текст окна) и поместить его числовой дескриптор во внутреннюю переменную.
/findwindowex(Button,Start)
Найти дочернее окно по (имя класса, текст окна) и поместить его числовой дескриптор во внутреннюю переменную. Дескриптор родительского окна должен быть во внутренней переменной.
/showwindow(5)
Показать окно. В параметрах указывается числовой код команды SW_. Дескриптор окна должен быть во внутренней переменной.
/setwindowtext(Text string)
Установить текст окна. Дескриптор окна должен быть во внутренней переменной.
/setwindowpos(200,200)
Установить позицию окна. В параметрах указывается X и Y координаты. Дескриптор окна должен быть во внутренней переменной.
/sendmessage(18,0,0)
Послать сообщение окну. Дескриптор окна должен быть во внутренней переменной.
/winamp(play)
Выполнить команду проигрывателя Winamp (имя команды в параметрах). Полный список команд см. в файле помощи.
/play(chimes.wav)
Проиграть звуковой файл. В параметрах команды укажите путь и имя файла.
/guid
Создает новый идентификатор класса (GUID) и помещает его в буфер обмена.
